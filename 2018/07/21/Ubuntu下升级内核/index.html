Ubuntu下更新内核版本

可以解决关机重启时间漫长的问题

  1. 一般内核有更新时会在终端显示有软件包可更新,执行命令更新软件包
    apt-get full-upgrade
  2. 更新后，更新grub系统引导文件，重启电脑
    sudo update-grub
sudo reboot
  3. 重启后，输入以下命令查看当前使用的内核
    uname -r

4.15.0-29-generic

4. 输入一下命令查看当前系统中安装的内核

sudo dpkg –get-selections |grep linux xiaoming@xiaoming-X542UN
