Comoposer的使用

发表于 2018-07-24 更新于 2020-12-01 阅读次数： 本文字数： 673 阅读时长 ≈ 1 分钟

composer用于安装相应的包

  1. 下载composer

    curl -sSh ttps://getcomposer.org/installer | php
    sudo mv composer.phar /usr/local/bin/composer

  • 在需要安装包的文件夹下创建.json文件
{
    "require":{
        //相应的包：相应的版本号
        //只能使用双引号
        "monolog/monolog":"1.2.*"
    }
}

需要什么包可以到packgist.org上查找

  • 在目录下执行composer install,将下载对应的包到vendor/monolog/monolog目录
  1. 在文件夹创建php脚本
<?php
   //引入自动加载文件
   require './vendor/autoload.php';
   
   //根据包的文档说明就使用工具包
?>
  1. 更新composer.json软件包
  • 修改.json文件
{
    "require":{
        //相应的包1：相应的版本号
        "monolog/monolog":"1.2.*",
        //相应的包2
        "monolog/monolog":"x.x.x"
    }
}
  • 执行composer update
  1. 更新composer：composer self-update
