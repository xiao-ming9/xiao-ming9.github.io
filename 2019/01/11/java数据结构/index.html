链表

线性表，但不会按线性的顺序存储数据，而是每一个节点里存着下一个节点的指针
链表和数组都是线性数据结构

  • 数组不便于做删除、插入，但遍历查找简单，有固定长度
  • 链表适合插入、删除，不宜过长，否则导致遍历性能下降
